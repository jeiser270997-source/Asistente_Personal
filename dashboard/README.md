# 🖥️ LifeOS Dashboard - Frontend Next.js

Este subproyecto contiene la interfaz gráfica del Segundo Cerebro de Jeiser. 
Permite visualizar de manera unificada el estado del SIMIT, postulaciones activas de Computrabajo, alertas de Zajuna (SENA), y bitácoras semánticas leídas directamente de SQLite.

## 🚀 Desarrollo

Arrancar el servidor de desarrollo local:
```bash
npm run dev
```

Construir para producción:
```bash
npm run build
npm start
```
